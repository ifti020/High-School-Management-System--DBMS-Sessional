# High School Management System

**High School Management System** is a web-based application developed using **PHP, MySQL, HTML, CSS, and Bootstrap**.  
This project was completed during the 2nd year as part of the **DBMS course** at BAUST.

---

## Project Overview

The system allows management of **students, teachers, and administrative tasks** in a high school.  
It is designed to provide an easy-to-use interface for admins, teachers, and students.

---

## Modules & Responsibilities

- **Admin Module** – Developed by **Nur**  
  Manage teachers, students, and overall school data.

- **Student Module** – Developed by **Ifti Haque**  
  View personal information, grades, and access notifications.

- **Teacher Module** – Developed by **Rakib**  
  Manage student grades, class assignments, and announcements.

---

## Technology Stack

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL
